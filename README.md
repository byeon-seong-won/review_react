<h1> 리뷰클릭 공통 컴포넌트 작업(React) </h1>


<h4>🔧 중점 둔 사항 🔧 <h4>
- map() 함수를 이용해 데이터 배열로 항목관리하여 반복되는 코드 제거
- react hook(useState)를 이용한 상태별 컴포넌트화
- 상태별 아이콘은 background-image로 넣어 제어
- 페이지 렌더링 시간 단축을 위해 태그 및 클래스 최소화, 올바른 css 선언순서
